# 配置系统

在编写扩展的时候，有可能需要保存一些配置，并提供一些配置让用户进行自定义设置。
所以编辑器内就提供了一套配置管理机制。

## 配置类型

编辑器里的配置分成了两种：

1. 编辑器设置
2. 项目设置

### 编辑器设置

存放一些编辑器相关的功能设置。这部分也是编辑器的主要配置。

编辑器设置分成三个层级，优先级从高到低：

```sh
local -> global -> default
```

### 项目设置

存放一些和项目相关的配置，这些配置允许，并且需要在项目间共享。

项目设置分成两个层级，优先级从高到低：

```sh
local -> default
```

关于如何扩展配置请查看[扩展配置](../contributions/profile/index.md)。
